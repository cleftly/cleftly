# Build

## Prerequisites

- [Rust & Cargo](https://rustup.rs/)
- [Node.js 20+](https://nodejs.org/en/)
- [Pnpm](https://pnpm.io/)

## 1. Clone the repository

```bash
git clone https://github.com/wxllow/cleftly.git
cd cleftly
```

## 2. Install dependencies

```bash
pnpm install
```

## 3. Run/Build

### Build

```bash
pnpm tauri build
```

### Dev Mode

```bash
pnpm tauri dev
```
